# 🏧 ATM Checker Project

A simple **Python-based ATM application** that allows users to securely log in and perform basic banking operations.

## ✨ Features
- 🔐 Login Authentication
- 💰 Deposit Money
- 💸 Withdraw Money
- 💳 Check Balance
- 🚪 Exit

## 🛠️ Technology
- 🐍 Python 3

## ▶️ How to Run
1. Open the project in VS Code or any Python IDE.
2. Run the Python file.
3. Enter your login credentials.
